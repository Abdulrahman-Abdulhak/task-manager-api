# Task Manager API

This is my first Node.js Express Server Application.

## Version

Version 1.0.0

## Features

- Create a Task
- Get All Tasks
- Get a Specific Task
- Delete a Specific Task
- Update a Specific Task

## HTTP Methods

- GET
- POST
- DELETE
- PATCH

## Installation and Running

Open the terminal and run the followings:

1. run "npm i"
2. run "npm run start"
